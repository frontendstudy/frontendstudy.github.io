CSS常见布局方式总结（下）

发表于 2015-10-20   |  

本文主要总结一下多列布局的常用方法和技巧。

一列定宽 一列自适应

实现效果

实现方法

1. 通过float和margin实现

<div class="parent">
    <div class="left">
        <p>left</p>
    </div>
    <div class="right">
        <p>right</p>
        <p>right</p>
    </div>
</div>  

.left{
    width:200px;
    float:left;
}
.right{
    margin-left:220px;          
}

这种方式实现是非常直观的，也是最容易想到的一种实现方式。IE6下有小的bug。

2. 通过float和overflow实现

<div class="parent">
    <div class="left">
        <p>left</p>
    </div>
    <div class="right">
        <p>right</p>
        <p>right</p>
    </div>
</div>  

.left{
    width:200px;
    float:left;
}
.right{
    overflow:hidden;
    margin-left:220px;          
}

这种方式实现是借助BFC的特性，使得元素触发BFC从而不与浮动元素重叠进行两列自适应布局。

3. 通过table实现

<div class="parent">
    <div class="left">
        <p>left</p>
    </div>
    <div class="right">
        <p>right</p>
        <p>right</p>
    </div>
</div>  

.parent{
    display: table;
    width:100%;
}
.parent div{
    display: table-cell;
}
.left{
    width:200px;
    padding-right:20px;
}
.right{
    /*这里什么也不用设置*/
}

这种方式的实现是借助了表格的特性，内部单元格要撑满table的宽度，从而达到了自适应的效果。但IE8以下是不支持的。

4. 通过flex实现

<div class="parent">
    <div class="left">
        <p>left</p>
    </div>
    <div class="right">
        <p>right</p>
        <p>right</p>
    </div>
</div>  

.parent{
    display: flex;
}
.left{
    width:200px;
}
.right{
    flex:1;
    margin-left: 20px;
}

flex的强大一直贯穿整个布局，实现也十分的优雅，只是低版本浏览器不支持，并且性能上会有一些损耗。

实现效果

要实现这样的效果，上述方法中只有方法1不可行，其他方法都是可行的，因为只有方法1的元素使用了left的宽度。

实现方法不再赘述。

等分布局

实现效果

实现方法

1. 通过float实现

<div class="parent">
    <div class="column">1</div>
    <div class="column">2</div>
    <div class="column">3</div>
    <div class="column">4</div>
</div>

.parent{
    margin-left: -20px;
}
.column{
    float:left;
    width:25%;
    padding-left:20px;
    box-sizing:border-box;
}

这种方法是通过浮动以及百分比控制的，实现的难点在于中间的固定间距，通过更改盒模型方式以及设置内边距可以达到固定间距的效果，其中首尾的间距可以通过父元素设置负的外边距来抵消。

2. 通过table实现

<div style="margin-left:-20px;">
    <div class="parent">
        <div class="column">1</div>
        <div class="column">2</div>
        <div class="column">3</div>
        <div class="column">4</div>
    </div>
</div>

.parent{
    display: table;
    width:100%;
}
.column{
    display: table-cell;
    padding-left: 20px;
}

这种方法利用table单元格默认等分的特性来实现效果，实现方法类似float的实现方式，只是由于table宽度计算的特殊性，还需要在加一层父元素来抵消间距。

3. 通过flex实现

<div class="parent">
    <div class="column">1</div>
    <div class="column">2</div>
    <div class="column">3</div>
    <div class="column">4</div>
</div>

.parent{
    display: flex;
}
.column{
    flex:1;
}
.column+.column{
    margin-left: 20px;
}

利用flex实现比较符合一般性思维，实现也比较简单，间距可以直接通过外边距来控制，不过还是存在兼容性和性能问题。

三列自适应布局

实现效果

实现方法

1. 通过float和margin实现

<div class="parent">
    <div class="left">
        <p>left</p>
    </div>
    <div class="center-help">
        <div class="center">
            <p>center</p>
        </div>
    </div>  
    <div class="right">
        <p>right</p>
    </div>
</div>  

.left{
    width:200px;
    float:left;
}
.center-help{
    float:left;
    width:100%;
    margin-left:-200px;
    margin-right: -200px;
}
.center{
    margin-left:220px;
    margin-right:220px;
}
.right{
    float:right;
    width:200px;
}

利用float和margin实现三栏自适应布局有些麻烦，必须要借助一个辅助结构实现中间的自适应布局，使得中间的结构抵消左右固定宽度，内容再通过外边距的设定实现固定的间距。

2. 通过float和overflow实现

<div class="parent">
    <div class="left">
        <p>left</p>
    </div>
    <div class="right">
        <p>right</p>
    </div>
    <div class="center">
        <p>center</p>
    </div>
</div>  

.left{
    width:200px;
    float:left;
}
.center{
    overflow:hidden;
    margin-left:220px;
    margin-right:220px;
}
.right{
    float:right;
    width:200px;
}

这个方法利用BFC的特性实现地十分优雅，只需要让center元素触发BFC从而实现不与左右两个浮动元素重叠，注意HTML的摆放顺序至关重要。

3. 通过table实现

<div class="parent">
    <div class="left">
        <p>left</p>
    </div>
    <div class="center">
        <p>center</p>
    </div>
    <div class="right">
        <p>right</p>
    </div>
</div>   

.parent{
    display: table;
    width:100%;
}
.parent div{
    display: table-cell;
}
.left{
    width:200px;
    padding-right:20px;
}
.center{
    padding-right:20px;
}
.right{
    width:200px;
}

利用table实现主要还是利用单元格要填充至table宽度的特性。注意table必须要给定宽度。

4. 通过position:absolute实现

<div class="parent">
    <div class="left">
        <p>left</p>
    </div>
    <div class="center">
        <p>center</p>
    </div>
    <div class="right">
        <p>right</p>
    </div>
</div>   

.parent{
    position: relative;
}
.parent div{
    position: absolute;
}
.left{
    width:200px;
}
.center{
    left:220px;
    right:220px;
}
.right{
    width:200px;
    right:0;
}

利用绝对定位实现比较好理解，也比较优雅。

5. 通过flex实现

<div class="parent">
    <div class="left">
        <p>left</p>
    </div>
    <div class="center">
        <p>center</p>
    </div>
    <div class="right">
        <p>right</p>
    </div>
</div>   

.parent{
    display:flex;
}
.left{
    width:200px;
}
.center{
    flex:1;
    margin-left:20px;
    margin-right:20px;
}
.right{
    width:200px;
}

flex总是如此强大和优雅。
