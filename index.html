CSS常见布局方式总结（上）

发表于 2015-10-20   |  

参加过各大公司的前端面试，发现很喜欢考布局方面的内容，基本上每次遇到这种问题都要想好久，所以在这里总结下关于这方面的知识。

水平居中布局

对于子元素和父元素都是固定宽度的情况，居中布局非常容易实现，直接margin:0 auto;即可。问题在于子元素和父元素的宽度都未知的情况下，该如何实现呢？

方法1：利用text-align和display:inline-block实现

方法思路： 对外层父元素设置text-align:center;；对内层子元素设置display:inline-block;即可实现效果。

实现原理： text-align会使得行内元素水平居中。

实现代码（HTML结构）

<div class="parent">
    <div class="child">child</div>
</div>

实现代码（CSS样式）

.parent{
    text-align: center;
}
.child{
    display: inline-block;
}

方法2：利用table和 margin:0 auto实现

方法思路： 对内层子元素设置display:table;margin:0 auto;即可实现效果。

实现原理： 根据table元素的特性。

实现代码（HTML结构）

<div class="parent">
    <div class="child">child</div>
</div>

实现代码（CSS样式）

.parent{
   /*父元素什么都不需要设置*/
}
.child{
    display: table;
    margin:0 auto;
}

方法3：利用absolute和 transform实现

方法思路： 父元素相对定位，子元素绝对定位，设置left值为50%，再拉回本身宽度的-50%.

实现原理： 绝对定位元素设置定位百分比相对于父级定位元素的宽度，translateX的偏移量相对于本身宽度。

实现代码（HTML结构）

<div class="parent">
    <div class="child">child</div>
</div>

实现代码（CSS样式）

.parent{
   position: relative;
}
.child{
    position: absolute;
    left:50%;
    transform:translateX(-50%);
}

方法4：利用flex实现

方法思路： 对父元素设置display:flex; justify-content:center;即可。

实现原理： 利用CSS3的flex弹性布局原理，可轻易控制元素的水平及垂直对齐方式。
实现代码（HTML结构）

<div class="parent">
    <div class="child">child</div>
</div>

实现代码（CSS样式）

.parent{
   display: flex;
   justify-content:center;
}
.child{
    /*子元素什么都不需要设置*/
}

垂直居中布局

垂直居中可不能用margin:auto 0;来实现，所以相对于水平居中来说实现难度稍微大了些，总体来说总结了相似的几种方法如下：

方法1：利用table-cell和vertical-align实现

方法思路： 对外层父元素设置display:table-cell; vertical-align:middle;；对内层子元素设置display:inline-block;即可实现效果。

实现原理： 单元格可以通过设置vertical-align来保证其内部的行内元素垂直居中。

实现代码（HTML结构）

<div class="parent">
    <div class="child">child</div>
</div>

实现代码（CSS样式）

.parent{
    display: table-cell;
    vertical-align: middle;
}
.child{
    display: inline-block;
}

方法2：利用absolute和 transform实现

方法思路： 父元素相对定位，子元素绝对定位，设置top值为50%，再拉回本身高度的-50%.

实现原理： 绝对定位元素设置定位百分比相对于父级定位元素的宽度，translateY的偏移量相对于本身高度。

实现代码（HTML结构）

<div class="parent">
    <div class="child">child</div>
</div>

实现代码（CSS样式）

.parent{
   position: relative;
}
.child{
    position: absolute;
    top:50%;
    transform:translateY(-50%);
}

方法3：利用flex实现

方法思路： 对父元素设置display:flex; align-items:center;即可。

实现原理： 利用CSS3的flex弹性布局原理，可轻易控制元素的水平及垂直对齐方式。

实现代码（HTML结构）

<div class="parent">
    <div class="child">child</div>
</div>

实现代码（CSS样式）

.parent{
   display: flex;
   align-items:center;
}
.child{
    /*子元素什么都不需要设置*/
}

水平垂直居中布局

水平居中布局的方式主要是通过以上两种布局的组合来实现，主要使用以下几种方法：

  • 利用inline-block+text-align+table-cell+vertical-align实现
  • 利用position+translate实现
  • 利用flex实现
