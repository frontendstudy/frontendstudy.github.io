常用事件小结

发表于 2015-10-24   |  

JS作为一门事件驱动的语言，其事件的重要性可想而知。常见的事件及其相关用法有些繁杂，在此总结一下，以备后用。

鼠标事件

  • click 单击事件
  • contextmenu 右击事件
  • dblclick 双击事件
  • mousedown 按下事件
  • mouseup 抬起事件
  • mousemove 移动事件
  • mouseover 移入事件
  • mouseout 移出事件
  • mouseenter 移入事件
  • mouseleave 移出事件
  • mousewheel 滚轮事件
  • DOMMouseScroll 滚轮事件

问：mousemove和mouseover事件有什么区别？

答：mouseover事件在鼠标移入到元素上时会触发，在元素内移动不会再次触发，mousemove事件当鼠标在元素上每移动一小段距离都会触发一次。

问：mouseover/mouseenter、mouseout/mouseleave有什么区别？

答：mouseover&mouseout默认是冒泡的，当给父元素添加此事件时，子元素的移入移出会再次触发。mouseenter&mouseleave是阻止冒泡的，一般来说后者更加符合我们的需求。

问：如何通过mousedown、mousemove、mouseup实现简单的拖拽功能？

答：大体上的思路是在mousedown时获得鼠标相对元素的位置，在mousemove时根据鼠标的偏移动态改变元素的位置，mouseup时取消move和up的事件。有三个关键点需要注意：

  1. 为了防止鼠标移动过快脱离元素造成拖动功能暂时失效，要将事件注册在document上。
  2. 为了防止父元素中有子元素时拖拽父元素，鼠标经过子元素会先触发子元素的mousemove造成混乱，需要将mousemove和mouseup都注册为捕获事件。
  3. 为了防止拖拽子元素时连同父元素也跟着走，需要阻止mousemove和mouseup的冒泡行为。
//不做兼容性处理
function drag(element) { 
    var deltaX, deltaY; //存储鼠标相对于元素的偏移位置。

    //注意这里，将事件注册在文档上。
    document.addEventListener('mousedown', downHandler);
    document.addEventListener('mousemove', moveHandler, true);
    document.addEventListener('mouseup', upHandler, true);

    function downHandler(e) {
        //这里假设元素绝对定位相对于body，否则需要精确计算其文档偏移量。
        //计算方法就是通过鼠标的文档坐标减去元素的文档坐标。
        deltaX = e.clientX + pageXOffset - element.offsetLeft;
        deltaY = e.clientY + pageYOffset - element.offsetTop;
    }

    function moveHandler(e) {
        element.style.left = e.clientX + pageXOffset - deltaX + 'px';
        element.style.top = e.clientY + pageYOffset - deltaY + 'px';
        e.stopPropagation();
    }

    function upHandler(e) {
        document.removeEventListener('mousemove', moveHandler, true);
        document.removeEventListener('mouseup', upHandler, true);
        e.stopPropagation();
    }
}

问：鼠标滚轮事件为什么有两个，它们的用法有什么不同？

答：一般滚轮事件是mousewheel，仅有firefox不支持它，所以只能用非标准的DOMMouseScroll来做兼容。在使用上也有区别，mousewheel事件对象上的wheelDelta能反映出滚动滚轮的情况，远离用户方向的一次滚动最小单位为120,接近的则为-120。

而对于DOMMouseScroll来说，利用事件对象上的detail属性反映滚轮情况。远离用户方向的一次滚动最小单位为-3,接近的则为3。

键盘与文本事件

  • keydown 按下事件
  • keyup 抬起事件
  • keypress 敲击事件
  • input 文本输入事件

问：input和key系列事件的本质区别是什么？

答：input事件上新输入的文本真正插入到聚焦的文档之后才触发，而key系列上插入到文档前触发。所以阻止keypress事件可以阻止用户的输入，input事件则可以用一些内容改变(全转换成大写)的场合。

问：keydown和keypress有什么区别？

答：可以简单地理解为keypress=keydown+keyup，但也并不完全是。比如一直按键不放时，keydown和keypress会交替连续触发。

另外，keypress常常不能显示组合键，一般在keypress不能捕获的时候使用keydown，keydown里面还有charCode属性，不过有兼容问题。

DnD拖放事件(Drag-and-Drop)（待续）

  • 任何有HTML draggable属性的文档元素都是拖放源。

1. 通过float和margin实现

3. 通过table实现

    </div>
    <div class="right">

.parent{
    display: table;
    width:100%;

.parent div{
    display: table-cell;
}
.left{
    width:200px;
    padding-right:20px;
}
.right{
    /*这里什么也不用设置*/
}

这种方式的实现是借助了表格的特性，内部单元格要撑满table的宽度，从而达到了自适应的效果。但IE8以下是不支持的。

4. 通过flex实现

<div class="parent">
    <div class="left">
        <p>left</p>
    </div>
    <div class="right">
        <p>right</p>
        <p>right</p>
    </div>
</div>  

.parent{
    display: flex;
}
.left{
    width:200px;
}
.right{
    flex:1;
    margin-left: 20px;
}

flex的强大一直贯穿整个布局，实现也十分的优雅，只是低版本浏览器不支持，并且性能上会有一些损耗。

等分布局

实现效果

实现方法

1. 通过float实现

<div class="parent">
    <div class="column">1</div>
    <div class="column">2</div>
    <div class="column">3</div>
    <div class="column">4</div>
</div>

.parent{
    margin-left: -20px;
}
.column{
    float:left;
    width:25%;
    padding-left:20px;

这种方法是通过浮动以及百分比控制的，实现的难点在于中间的固定间距，通过更改盒模型方式以及设置内边距可以达到固定间距的效果，其中首尾的间距可以通过父元素设置负的外边距来抵消。

2. 通过table实现

<div style="margin-left:-20px;">
    <div class="parent">
        <div class="column">1</div>
        <div class="column">2</div>
        <div class="column">3</div>
        <div class="column">4</div>
    </div>
</div>

.parent{
    display: table;
    width:100%;
}
.column{
    display: table-cell;
    padding-left: 20px;
}

这种方法利用table单元格默认等分的特性来实现效果，实现方法类似float的实现方式，只是由于table宽度计算的特殊性，还需要在加一层父元素来抵消间距。

3. 通过flex实现

<div class="parent">
    <div class="column">1</div>
    <div class="column">2</div>
    <div class="column">3</div>
    <div class="column">4</div>
</div>

.parent{
    display: flex;
}
.column{
    flex:1;
}
.column+.column{
    margin-left: 20px;
}

利用flex实现比较符合一般性思维，实现也比较简单，间距可以直接通过外边距来控制，不过还是存在兼容性和性能问题。

CSS常见布局方式总结（上）

发表于 2015-10-20   |  

参加过各大公司的前端面试，发现很喜欢考布局方面的内容，基本上每次遇到这种问题都要想好久，所以在这里总结下关于这方面的知识。

水平居中布局

对于子元素和父元素都是固定宽度的情况，居中布局非常容易实现，直接margin:0 auto;即可。问题在于子元素和父元素的宽度都未知的情况下，该如何实现呢？

方法1：利用text-align和display:inline-block实现

方法思路： 对外层父元素设置text-align:center;；对内层子元素设置display:inline-block;即可实现效果。

实现原理： text-align会使得行内元素水平居中。

实现代码（HTML结构）

<div class="parent">
    <div class="child">child</div>
</div>

实现代码（CSS样式）

.parent{
    text-align: center;
}
.child{
    display: inline-block;
}

方法2：利用table和 margin:0 auto实现

方法思路： 对内层子元素设置display:table;margin:0 auto;即可实现效果。

实现原理： 根据table元素的特性。

实现代码（HTML结构）

<div class="parent">
    <div class="child">child</div>
</div>

实现代码（CSS样式）

.parent{
   /*父元素什么都不需要设置*/
}
.child{
    display: table;
    margin:0 auto;
}

方法3：利用absolute和 transform实现

方法思路： 父元素相对定位，子元素绝对定位，设置left值为50%，再拉回本身宽度的-50%.

实现原理： 绝对定位元素设置定位百分比相对于父级定位元素的宽度，translateX的偏移量相对于本身宽度。

实现代码（HTML结构）

<div class="parent">
    <div class="child">child</div>
</div>

实现代码（CSS样式）

.parent{
   position: relative;
}
.child{
    position: absolute;
    left:50%;
    transform:translateX(-50%);
}

方法4：利用flex实现

方法思路： 对父元素设置display:flex; justify-content:center;即可。

实现原理： 利用CSS3的flex弹性布局原理，可轻易控制元素的水平及垂直对齐方式。

实现代码（HTML结构）

<div class="parent">
    <div class="child">child</div>
</div>

实现代码（CSS样式）

.parent{
   display: flex;
   justify-content:center;
}
.child{
    /*子元素什么都不需要设置*/
}

垂直居中布局

垂直居中可不能用margin:auto 0;来实现，所以相对于水平居中来说实现难度稍微大了些，总体来说总结了相似的几种方法如下：

方法1：利用table-cell和vertical-align实现

方法思路： 对外层父元素设置display:table-cell; vertical-align:middle;；对内层子元素设置display:inline-block;即可实现效果。

实现原理： 单元格可以通过设置vertical-align来保证其内部的行内元素垂直居中。

实现代码（HTML结构）

<div class="parent">
    <div class="child">child</div>
</div>

实现代码（CSS样式）

.parent{
    display: table-cell;
    vertical-align: middle;
}
.child{
    display: inline-block;
}

方法2：利用absolute和 transform实现

方法思路： 父元素相对定位，子元素绝对定位，设置top值为50%，再拉回本身高度的-50%.

实现原理： 绝对定位元素设置定位百分比相对于父级定位元素的宽度，translateY的偏移量相对于本身高度。

实现代码（HTML结构）

<div class="parent">
    <div class="child">child</div>
</div>

实现代码（CSS样式）

.parent{
   position: relative;
}
.child{
    position: absolute;
    top:50%;
    transform:translateY(-50%);
}

方法3：利用flex实现

方法思路： 对父元素设置display:flex; align-items:center;即可。

实现原理： 利用CSS3的flex弹性布局原理，可轻易控制元素的水平及垂直对齐方式。

实现代码（HTML结构）

<div class="parent">
    <div class="child">child</div>
</div>

实现代码（CSS样式）

.parent{
   display: flex;
   align-items:center;
}
.child{
    /*子元素什么都不需要设置*/
}

水平垂直居中布局

水平居中布局的方式主要是通过以上两种布局的组合来实现，主要使用以下几种方法：

  • 利用inline-block+text-align+table-cell+vertical-align实现
  • 利用position+translate实现
  • 利用flex实现
